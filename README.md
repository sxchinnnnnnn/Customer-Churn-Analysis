# Customer-Churn-Analysis
Customer churn prediction system using ML and Power BI. Built with Random Forest, deployed via Streamlit and Flask API, providing real-time churn insights and interactive dashboards.

# 📊 Customer Churn Prediction & Analytics

## 🚀 Project Overview

This project is an end-to-end Customer Churn Analysis system that combines:

* Data Analysis (Python)
* Machine Learning (Random Forest)
* Power BI Dashboard
* Streamlit Web App
* Flask API

## 🎯 Objective

To identify customers likely to churn and provide business insights to improve retention.

---

## 🧠 Machine Learning

* Model: Random Forest Classifier
* Accuracy: ~85%
* Features: tenure, contract, charges, payment, support

---

## 🌐 Deployment

* Streamlit app for UI-based predictions
* Flask API for backend predictions

---

## 📊 Power BI Dashboard

Includes:

* Executive Overview
* Churn Drivers
* Revenue Impact
* ML Prediction Insights

---

## 🛠️ Tech Stack

* Python (Pandas, Scikit-learn)
* Power BI
* Streamlit
* Flask

---

## ▶️ How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Run Streamlit app

```
streamlit run streamlit_app.py
```

### 3. Run Flask API

```
python flask_api.py
```

---

## 💼 Key Insights

* Month-to-month customers have highest churn
* High charges increase churn probability
* Lack of support services increases churn risk
